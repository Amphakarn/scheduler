# Interview Scheduler

Interview Scheduler is a single-page application that was built using React and test with the latest tools and techniques. This application allows users to book, cancel and edit interviews. 


## Dependencies

- axios
- classnames
- normalize.css
- react
- react-dom
- react-scripts
